# 0069_关系运算符使用

- 细节说明
1. 关系运算符的结果都是boolean型，也就是要么是true，要么是false。
2. 关系运算符组成的表达式，我们称为关系表达式。 a > b。
3. 比较运算符 "==" 不能误写成 ”=”
