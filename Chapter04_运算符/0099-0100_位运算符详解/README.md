# 0099-0100_位运算符详解

- java中有7个位运算
    - &、|、^、~、>>、<<、>>>
- 运算法则：
    - 按位与 &：两位全为1，结果为1，否则为0。
    - 按位或 |：两位有一个为1，结果为1，否则为0。
    - 按位异或 ^：两位一个为0，一个为1，结果为1，否则为0。
    - 按位取反 ~：0 → 1，1 → 0。
    - 算数右移 >>：低位溢出，符号位不变，并用符号位补溢出的高位。
    - 算数左移 <<：符号位不变，低位补0。
    - 逻辑右移 >>>：也叫无符号右移，低位溢出，高位补 0 。
    - 特别说明：没有 <<< 符号！
- 计算：
    - 2 & 3 = 2
    - ~(-2) = 1
    - ~2 = -3
    - 1 >> 2 = 0 [本质：1 / 2 / 2 = 0]
    - 1 << 2 = 4 [本质：1 * 2 * 2 = 4]
    - 4 << 3 = 32 [本质：4 * 2 * 2 * 2 = 32]
    - 15 >> 2 = 3 [本质：15 / 2 / 2 = 3]
